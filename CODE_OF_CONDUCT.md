# Code of Conduct

This project expects respectful, technical, and constructive collaboration.

## Expected Behavior

- Assume good intent, but discuss code and decisions directly.
- Give actionable feedback tied to behavior, code, or documentation.
- Respect different experience levels and communication styles.
- Keep security-sensitive reports private.

## Unacceptable Behavior

- Harassment, personal attacks, or discrimination
- Deliberate disruption of discussions or reviews
- Publishing private or security-sensitive information without consent

## Scope

This applies to issues, pull requests, discussions, and other project spaces.

## Reporting

Report conduct issues privately to the project maintainers through GitHub's maintainer contact channels. Public threads should not be used for sensitive reports.
